
            Sample for Virtual Cable Test
====================================================

This sample shows how to run VCT(Virtual Cable Tester), Advance VCT and how to get
1000B-T extended status, such as pair swap, pair skew, and polarity.

Main Functions:
    madVctTest           - run VCT and display the result.
    madAdvVctTest        - run Advanced VCT and display the result.
    madGetExtendedStatus - get 1000Base-T Extended Status.

Used APIs:
    mdDiagGetCableStatus,
    mdDiagGetAdvCableStatus,
    mdDiagGet1000BTExtendedStatus
