
         Sample for External Loopback and MAC Interface Loopback setup
==============================================================================

This sample shows how to start/stop External Looback and MAC Interface Loopback.

Main Functions:
    startExLoopback - start External Loopback
    stopExLoopback  - stop External Loopback
    startMACIfLoopback - start MAC Interface Loopback
    stopMACIfLoopback  - stop MAC Interface Loopback

Used APIs:
    mdCopperGetAutoNeg,
    mdCopperSetAutoNeg,
    mdDiagSetExternalLoopback,
    
    mdCopperGetMDIMode,
    mdCopperSetMDIMode,
    mdSGMIIGetAutoNeg,
    mdSGMIISetAutoNeg,
    mdDiagSetMACIfLoopback
