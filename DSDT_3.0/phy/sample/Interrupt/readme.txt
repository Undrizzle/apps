
            Sample for Interrupt related APIs
=======================================================

This sample shows how to enable/disable interrupts and how to read/handle interrup
causes.

Main Functions:
    sampleMADIntEnable  - enable Interrupts
    sampleMADIntDisable - disable Interrupts
    sampleMADIntHandler - read and handle Interrupts

Used APIs:
    mdIntSetEnable,
    mdIntGetPortSummary,
    mdIntGetStatus
